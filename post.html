Salary prediction Project

Prediction Of Employee Salary on The Bases of Previous Company Data With Linear Regression.

Project Objective

The HR team of a company uses to determine what salary to offer to a new employee. let's take an example that an employee has applied for the role of a Data Engineer and has already worked as a SDE for 2 years. So based on the data provided we want to build a model to predict what salary we should offer to new employee if we come to know the true salary from previous company.

This model is implemented by using the following steps such as :-

1. Import all required libraries like pandas,numpy etc

2. load data

3. perform Exploratory Data Analysis(EDA)

4. visualize data and get some valuable outputs from it

5. prepare data: It includes Data preprocessing part where we need to check Null or unique values and some statistical description of the dataset

6. split data into training and testing: it includes training and splitting of the model using test data and train data

7. define model: here we need to define our model which is best suitable for our training data I used linear regression

8. test model: we can check the prdictions of the model with our testing data

9. check accuracy with some other learning algorithms which is more efficient and also the problem of overfitting and errors with mean squared error method
